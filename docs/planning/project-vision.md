 Project Vision — Sistema de Monitoramento Inteligente com ESP32

 Nome do Projeto
SmartSense IoT

 Problema que resolve
Atualmente, o monitoramento de variáveis ambientais (como temperatura e umidade) é feito de forma manual ou com sistemas pouco acessíveis, dificultando a identificação de anomalias em tempo real.

Isso pode gerar prejuízos em ambientes comerciais, industriais e até residenciais.

 Objetivo principal
Desenvolver um sistema inteligente utilizando ESP32 capaz de coletar, enviar e analisar dados ambientais em tempo real, permitindo monitoramento remoto e identificação de anomalias.

  Público-alvo
- Pequenos comércios
- Ambientes industriais
- Residências inteligentes
- Estudantes e projetos acadêmicos

  Benefícios
- Monitoramento em tempo real
- Acesso remoto via dashboard
- Alertas automáticos
- Histórico de dados
- Base para análise inteligente (Machine Learning)

  Funcionalidades principais
- Coleta de dados com ESP32 + sensores
- Envio de dados via API (JSON)
- Armazenamento em banco de dados
- Dashboard com visualização em tempo real
- Sistema de alertas
- Histórico de medições

 Escopo inicial (MVP)
- Leitura de temperatura e umidade (DHT22)
- Envio de dados para API
- Armazenamento em banco
- Exibição simples dos dados

 Escopo futuro
- Machine Learning para detecção de anomalias
- Notificações via WhatsApp ou e-mail
- Suporte a múltiplos sensores
- Aplicativo mobile
- Controle automático de dispositivos (ex: ligar ventilação)
