
Roadmap — SmartSense IoT

 Bimestre 1 — Planejamento
- Definição do projeto
- Criação do repositório no GitHub
- Estruturação da documentação
- Definição das tecnologias

Bimestre 2 — Desenvolvimento Técnico
- Configuração do ESP32
- Integração com sensor DHT22
- Envio de dados via Wi-Fi
- Desenvolvimento da API
- Criação do banco de dados



 Bimestre 3 — Finalização
- Desenvolvimento do dashboard
- Testes do sistema completo
- Ajustes finais
- Deploy da versão 1.0
- Apresentação do projeto



 Objetivo Final
Entregar um sistema funcional de monitoramento inteligente com coleta, envio e visualização de dados em tempo real.
